# 🚗 Featured Project

## 4RunnerDash

**Repository:**
https://github.com/TB543/4RunnerDash

4RunnerDash is my flagship personal project—a **modular dashboard system** designed for the Toyota 4Runner. Rather than being a single-purpose application, it serves as a platform capable of hosting multiple independent applications through a unified interface.

The project places a strong emphasis on both **software architecture** and **UI/UX**. Built with **CustomTkinter**, the dashboard features a fully custom interface designed to be clean, intuitive, and easily expandable. Every component was designed with usability in mind while maintaining a modular architecture that allows new applications to be integrated without modifying the core dashboard.

### Features

* Modular application framework
* Custom-built desktop UI
* Designed for extensibility
* Hardware-focused architecture
* Clean, responsive interface
* Easy integration of additional applications

### Included Applications

#### 📁 4RunnerExplorer

Repository:
https://github.com/TB543/4RunnerExplorer

A fully integrated file explorer built specifically for the 4RunnerDash ecosystem. It demonstrates how independent applications can seamlessly plug into the dashboard while maintaining a consistent user experience.

#### 🎸 4RunnerEffects

Repository:
https://github.com/TB543/4RunnerEffects

A virtual guitar pedalboard application that turns the dashboard into a portable effects processor. The project includes a custom interface for configuring pedals and demonstrates real-time audio processing along with extensive custom UI development using CustomTkinter.
